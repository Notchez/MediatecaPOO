# Sistema de Mediateca

Aplicación de escritorio desarrollada en Java utilizando Swing y MySQL.

## Funcionalidades
- Gestión de Libros
- Gestión de Revistas
- Gestión de CD Audio
- Gestión de DVD
- Operaciones CRUD completas
- Registro de eventos y errores con Log4J

## Tecnologías
- Java
- Swing
- MySQL
- JDBC
- Log4J

## Base de datos
Ejecutar el archivo `mediateca.sql` para crear la base de datos.

## Ejecución
1. Configurar conexión en `Conexion.java`
2. Ejecutar la clase `Main`
## Librerías incluidas

Las dependencias necesarias ya se encuentran en la carpeta `lib`.

Si IntelliJ no las reconoce automáticamente:
1. Ir a File → Project Structure
2. Añadir los archivos `.jar` desde la carpeta `lib`
